# Airbnb Analysis Dashboard

## Preview

The dashboard includes pages for:

- Airbnb market overview
- City performance
- Pricing analysis

Open `Gold layer/PowerBI/Airbnb.pbip` in Power BI Desktop to view or edit the final dashboard.
